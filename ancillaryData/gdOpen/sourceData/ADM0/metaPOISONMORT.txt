geoDesc (www.geodesc.org) Ancillary Variables Metadata File 
==============================================================
Metadata for: POISONMORT
Full Variable Name: Mortality rate attributed to unintentional poisoning
==============================================================

Data Definition: Mortality rate attributed to unintentional poisonings is the number of deaths from unninstentional poisonings in a year per 100,000 population.  Unintentional poisonings can be caused by household chemicals, pesticides, kerosene, carbon monoxide and medicines, or can be the result of environmental contamination or occupational chemical exposure.
Units: Deaths per 100,000 population
Data Representative Of Year(s): 2016-2016
geoBoundary Shapefile Dataset API Link: https://www.geoboundaries.org/gbRequest.html?ISO=GLB&VER=3_0_0&ADM=ADM0
Each row ID is defined by: ISO 3166-1 alpha-3
Data Processing Methodology: Retrieved from World Bank World Development Indicators Website on October 14, 2020.
Data Collection Methodology (or link to documentation): Compiled by the World Health Organization, the data in this dataset come from official death registration data where available (see notes).
